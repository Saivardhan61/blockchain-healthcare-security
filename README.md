# Blockchain-Driven Security Framework for Healthcare Record Management

## Overview
This project is a blockchain-driven healthcare record management system developed using Java technologies. The application provides secure storage, management, and verification of healthcare records with enhanced security features.

## Features
- Secure healthcare record management
- Blockchain-based security framework
- Patient and doctor authentication
- Secure cloud data verification
- Medical record access management
- Data privacy and protection

## Technologies Used
- Java
- JSP
- Servlets
- HTML
- CSS
- JavaScript
- MySQL

## Project Structure
- src/main/java
- src/main/webapp

## How to Run
1. Import project into Eclipse
2. Configure Apache Tomcat Server
3. Configure MySQL Database
4. Run the project on server

- AI-based healthcare analysis
- Enhanced blockchain integration
- Advanced encryption methods
